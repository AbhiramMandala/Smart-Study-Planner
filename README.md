# 📚 Smart Study Planner

A simple yet powerful Python-based study planning application designed to help students organize subjects, track study progress, and prepare effectively for exams.

🚀 Overview

Smart Study Planner helps students create structured study schedules by managing subjects, required study hours, and progress tracking. The project demonstrates fundamental Python programming concepts including file handling, dictionaries, functions, loops, and user interaction.

This project was developed as a final project for Code in Place 2026.

---

✨ Features

- ➕ Add and manage subjects
- ⏳ Track required study hours
- 📅 Generate personalized study plans
- 📈 Monitor study progress
- 💾 Save progress automatically
- 📊 View overall study statistics
- 🔄 Persistent data storage using JSON

---

🛠️ Technologies Used

- Python 3
- JSON File Handling
- Command Line Interface (CLI)
